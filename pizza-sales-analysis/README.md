# Pizza Sales Analysis  
Power BI Dashboard and Python EDA Project

## Project Overview

This project presents an end-to-end Pizza Sales Analysis using Python for data cleaning and exploratory data analysis, and Power BI for building an interactive business dashboard.

The objective of this project is to analyze sales performance, identify top and bottom selling products, and uncover customer ordering patterns to generate actionable business insights.

---

## Key Business Metrics

- Total Revenue: 313K+
- Total Orders: 8,200+
- Total Pizzas Sold: 19,000+
- Average Order Value: 38+
- Average Pizzas per Order: 2.32

---

## Key Insights

- The Classic category contributes the highest sales.
- Large-sized pizzas generate maximum revenue.
- Orders peak on Friday and Saturday evenings.
- January and March recorded the highest monthly sales.
- Identified top and bottom performing pizzas based on revenue and quantity.

---

## Python Analysis (Exploratory Data Analysis)

The Jupyter Notebook includes:

- Data Cleaning
- Handling Missing Values
- Feature Engineering
- Sales Trend Analysis
- Category and Size Analysis
- Data Visualization using Matplotlib and Seaborn
